# Bienvenue sur RESERVIA

Ce projet consiste à implémanter une maquette d'un site internet de réservation d'hotel en ligne, avec HTML et CSS.

### Techniques de développement :

- Sémantique HTML
- CSS FlexBox
- CSS Grid
- Responsive


### Outils techniques :

- CSS
- HTML
